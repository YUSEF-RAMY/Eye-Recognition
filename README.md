# рҹ“Ң Overview

The **Eye Recognition System** is a biometric authentication solution designed to identify individuals using unique eye patterns.

This system combines **Artificial Intelligence, Image Processing, and Web/Mobile technologies** to provide a secure and accurate identity verification mechanism.

The project was developed as a **Final Graduation Project (MET)** and demonstrates the integration of AI models with real-world applications.

# рҹ§  AI Recognition Workflow

The AI model processes eye images through several stages:

1пёҸвғЈ Eye Image Capture
2пёҸвғЈ Image Preprocessing
3пёҸвғЈ Feature Extraction
4пёҸвғЈ Embedding Generation
